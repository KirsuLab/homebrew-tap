# KirsuLab Homebrew Tap

Homebrew tap for [KirsuLab](https://kirsulab.com) macOS apps.

## Conduit — Android file manager for Mac

```sh
brew install --cask KirsuLab/tap/conduit
```

Conduit lets you browse, copy, upload and delete your Android phone's files right from your Mac, over USB or Wi-Fi. More: https://kirsulab.com/macos/conduit
